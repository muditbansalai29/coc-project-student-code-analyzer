#  Student Score Analyzer

##  Project Overview
This is a simple C program that helps analyze student quiz scores. It allows the user to input multiple scores, then calculates and displays:
- The average score
- The highest score
- The lowest score

##  Concepts Used

### C Programming
- Arrays
- Loops (`for`)
- Functions
- Conditional statements (`if`)
- Basic I/O (`scanf`, `printf`)

### Mathematics
- Mean (Average)
- Minimum and Maximum values
